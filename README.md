# 文章列表爬蟲 Blog Article Crawler

### Description

爬文章列表，儲存所有部落格文章至 Docx 文件

Crawl article list to docx file

### Usage

```bash
ruby app.py BLOG_URL
```

### Example Input

in ./save/SITE_DOMAIN/

### Output

All the output files will be in ./save directory,

and all docx files are in ./save/SITE_DOMAIN/
